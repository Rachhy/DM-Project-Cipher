II.I. RSA Cryptography
   "RSA (Rivest-Shamir-Adleman) is a widely used asymmetric encryption algorithm that relies on the mathematical properties of prime numbers. It was invented in 1977 by Ron Rivest, Adi Shamir, and Leonard Adleman. RSA is based on the difficulty of factoring large composite numbers into their prime factors."
II.I.I RSA Keygen

   RSA Key Generation is the process of creating a pair of keys: a public key and a private key. These keys are mathematically related and are used for encryption and decryption.

Select Two Prime Numbers

Choose two large prime numbers:

  • ( p )
  • ( q )

These prime numbers must be kept secret.

Example:

p = 61 , q = 53

Compute ( N )

Multiply the two primes:

N = p × q = 61 × 53 = 3233, The value ( N ) is used in both public and private keys.

Compute Eulers Totient Function

φ(N) = (p - 1)(q - 1) ==> φ(N) = 60 × 52 = 3120

Choose the Public Exponent ( e )

Select a number ( e ) such that:

  • 1 < e < φ(N)
  • gcd(e, φ(N)) = 1, Example: e = 7

Compute the Private Exponent ( d )

Find ( d ) such that: ( d × e = 1 (mod φ(N)) )

Result : d = 1783

Form the Key Pair

  • Public Key: (e, N) = (7, 3233)
  • Private Key: (d, N) = (1783, 3233)
  • Public key is used for encryption
  • Private key is used for decryption
  • Security depends on the difficulty of factoring ( N ) into ( p ) and ( q )
