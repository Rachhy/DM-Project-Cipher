I.I. Caesar Cipher : A Secret Message from Ancient Rome
    "One of the earliest and most well-known forms of cryptography was developed by Emperor Julius Caesar. During times of war, he needed a way to send secret messages to his generals. His solution was simple but effective: shift each letter in the alphabet by three positions. For example, the letter A becomes D, B becomes E, and so on. This technique came to be known as the Caesar Cipher."
With a shift value of k = 3, a readable command became a coded secret — clear to allies, but meaningless to enemies.
Image : Caesar Cipher Wheel

Creating a Secret Alphabet Code

Each letter of the alphabet is first replaced with a number:

A = 00, B = 01, C = 02, D = 03, E = 04, F = 05, G = 06, H = 07, I = 08, J = 09, K = 10, L = 11, M = 12, N = 13, O = 14, P = 15, Q = 16, R = 17, S = 18, T = 19, U = 20, V = 21, W = 22, X = 23, Y = 24, Z = 25

Turning the Message into Numbers

Suppose the secret message is: SOTHUNRACHHY

S O T H U N R A C H H Y
18 14 19 07 20 13 17 00 02 07 07 24

Shifting the Letters (k = 3)

Caesar's rule is simple: add 3 to every number.

Before, Shifting : SOTHUNRACHHY = 181419072013170002070724

18+3 , 14+3 , 19+3 , 07+3 , 20+3 , 13+3 , 17+3 , 00+3 , 02+03 , 07+3 , 07+3 , 24+3
= 21, 17, 22, 10, 23, 16, 20, 03, 05, 10, 10, 27

Wrapping Around with Modulo 26

Because the alphabet contains only 26 letters, any value greater than 25 must be wrapped around using modulo 26.

27 mod 26 = 1

So, (27 = 01 = B).It works the same fior every Number.

Final encrypted numbers:
21 17 22 10 23 16 20 03 05 10 10 27

Revealing the Ciphertext

Converting the encrypted numbers back into letters gives:

21 = V, 17 = R, 22 = W, 10 = K, 23 = X, 16 = Q, 20 = U, 03 = D, 05 = F, 10 = K, 10 = K, 27 = B

Final Encrypted Message

Ek=3(SOTHUNRACHHY) = VRWKXQUDFKKB

( To decrypt the message, simply reverse the process by subtracting 3 from each letter's number and applying modulo 26 again. This simplicity is what made the Caesar Cipher easy to use—yet effective for its time.)

I.II. General Shift Cipher
    "The General Shift Cipher is a more advanced version of the Caesar Cipher. Instead of a fixed shift of three positions, it allows for any integer value as the shift. This means that you can choose any number to shift the letters in the alphabet, making it more versatile and secure."

I.III. Affine Cipher
    "The Affine Cipher is a more complex encryption method that combines both a shift and a multiplication operation. It uses two keys: one for the shift and another for the multiplication. The formula for encryption is E(x) = (ax + b) mod 26, where 'a' and 'b' are the keys, and 'x' is the numerical representation of the letter."

Affine Cipher Demo

The Affine Cipher is an encryption method that combines multiplication and shift operations. It uses two keys: a and b.

We have :

A = 00, B = 01, C = 02, D = 03, E = 04, F = 05, G = 06, H = 07, I = 08, J = 09, K = 10, L = 11, M = 12, N = 13, O = 14, P = 15, Q = 16, R = 17, S = 18, T = 19, U = 20, V = 21, W = 22, X = 23, Y = 24, Z = 25

Encryption formula:

E(x) = (a * x + b) mod 26

Example :

Keys: a = 5, b = 8

  1. H → 7
  2. E(7) = (5 * 7 + 8) mod 26 = 17
  3. 17 → R

Result: H → R

I.IV. Transposition Cipher

    Step into the fascinating world of cryptography, where transposition ciphers reveal the clever art of hiding messages. As one of the oldest and simplest encryption methods, transposition ciphers work by shuffling the positions of characters rather than changing the characters themselves. Unlike substitution ciphers, every letter stays the same—only the order is rearranged. This clever twist turns a readable message into a mysterious puzzle.

Example of Transposition Cipher using key = 4.

Plaintext: HELLO WORLD

First, remove the space to get HELLOWORLD.

Since the key is 4, write the text row by row into 4 columns.

H E L L
O W O R
L D X X

Next, read the letters column by column.

Column 1: H O L
Column 2: E W D
Column 3: L O X
Column 4: L R X

The encrypted text is: HOLEWDLOXLRX
To decrypt, write the encrypted text column by column using the same key.
Then read the letters row by row to get: HELLOWORLDXX.
Remove the extra X characters to obtain the original message: HELLOWORLD.
